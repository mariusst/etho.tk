About

etho.tk is an online project created by Marius Stoica in collaboration with Nita Mocanu. The aim is to create and to present theoretical and artistic works at the crossroads of contemporary art, contemporary philosophy (especially speculative realism and experimental philosophy) and technology (especially semantic web technology). The source code for etho.tk can be found on GitHub
